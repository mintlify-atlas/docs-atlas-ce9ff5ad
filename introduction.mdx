## What is Yoko?

Yoko is a Python-based chatbot that connects to WhatsApp via the Meta Cloud API. When a student sends a message, Yoko:

1. Receives the message through a Flask webhook
2. Searches a structured knowledge base for relevant context
3. Sends that context along with the question to the Groq LLM (llama3-8b-8192)
4. Returns a natural-language answer directly to the student on WhatsApp

It is designed for university support scenarios — answering questions about schedules, enrollment, administrative procedures, and other campus-related topics.

## How it works

```
Student (WhatsApp)
      │
      ▼
Meta Cloud API  ──POST──▶  /webhook  (app.py)
                                │
                                ▼
                         mcp.py: buscar_contexto()
                         — fuzzy-match question against data.json
                                │
                                ▼
                         llm.py: consultar_llm()
                         — send (context + question) to Groq API
                                │
                                ▼
                         app.py: enviar_mensaje()
                         — reply to the student via WhatsApp API
```

## Project structure

| File | Purpose |
|---|---|
| `app.py` | Flask server — webhook endpoint, message dispatch |
| `mcp.py` | Context retrieval — fuzzy-matches questions against `data.json` |
| `llm.py` | LLM client — sends prompts to Groq and returns answers |
| `data.json` | Knowledge base — Q&A pairs for your university |
